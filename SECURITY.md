# 🔐 Política de Seguridad (Security Policy)

## Versiones Soportadas (Supported Versions)

Actualmente solo se brindan actualizaciones de seguridad a la última versión estable del dashboard.

| Versión | Soportado          |
| ------- | ------------------ |
| 2.x     | ✅ Sí               |
| < 2.0   | ❌ No               |

## Reportar una Vulnerabilidad (Reporting a Vulnerability)

La seguridad de este proyecto es prioridad. Si encuentras una vulnerabilidad, por favor **NO la reportes públicamente** abriendo un issue.

En su lugar, por favor sigue estos pasos:
1. Envía un correo electrónico detallado a **seguridad@wallbitdash.io** (o contacta directamente a Johann via GitHub).
2. Describe el problema y cómo reproducirlo.
3. Danos un tiempo razonable para investigar y solucionar el problema antes de hacerlo público.

## Proceso de Respuesta

Una vez recibido el reporte:
- Confirmaremos la recepción del mismo en un plazo de 48-72 horas.
- Trabajaremos en un parche o mitigación.
- Te mantendremos informado sobre el progreso y el crédito correspondiente (si lo deseas).

Gracias por ayudar a mantener este proyecto seguro. ✨
